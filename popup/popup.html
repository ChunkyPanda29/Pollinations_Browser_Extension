Quick Guide 🎓

1. Setup & API Keys

Get your key at enter.pollinations.ai. Ensure you have credits (Pollen). 💎 Models require paid pollen.

2. How to Use

1. Select Mode: Image, Video, or Text.

2. Input: Type a prompt or upload reference images.

3. Context: Highlight text or right-click images on any website and select "Send to Pollinations".

4. Generate & Download: Click Generate, wait for the result, then click Download to save.

3. Advanced Tips

• Use "Custom Size" for specific aspect ratios.

• Use a specific Seed number to reproduce results.

• You can upload up to 4 reference images.
